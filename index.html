Create a New Repository

    1. Enter a unique repository name
    2. Enter some description about repository (optional)
    3. Make repository public / private
    4. Add Readme.md or other files (optional)
    5. Click on create repository

Get HTTP Link

    1. After creating repository you will redirect another page
    2. Click HTTPS option
    3. Copy generated link

Clone to Local Repository

    1. Now open terminal
    2. And clone the repository into local repository
    3. Type git clone [HTTPS link that you copied from website] & press enter

Create Required Files

    1. Open VS Code editor in the same folder
    2. Create the required files (e.g. .html/.css)
    3. Save the file's in same repository

Check Git Status

    1. Check status of local repository
    2. Type git status
    3. File names in red, shows that files are not added to git yet

Adding Files to Local Repository

    1. To add unadded files to local repository use git add
    2. We can add files in two ways
      .  Add some specific file by git add 'file name'
      .  Add all files by git add
    3. Now if we write git status, we can see all red files turned into green, it means we successfully added these files into Local Repository

Commit Files Locally

    1. Use git commit to commit files locally
    2. Type git commit -m "message", we can write any message that we want
      .  Message should describe the working of the commit

Check Origin

    1. To check origin use git remote -v
    2. If the Origin isn't added, then add origin by git remote add origin [HTTPS link]
      .  HTTPS link should be same that we use to Clone Git Repository

Push Code to Remote Repository

    1. To push the code to remote repository use git push origin [branch name]
    2. After git push origin [branch name] terminal will get your github username and password
    3. After push, Code will be published on your Github Repository
